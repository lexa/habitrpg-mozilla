Habitica Extension welcome page

Hello. Thank you for installing Habitica extension for firefox.
To make this addon work, please fill "User Id" and "API key" settings in addon preferences. You can find this values here Habitica.com->settings->API . Please not that Used ID and API key are NOT your login/password which you use to have access to Habitica website.
